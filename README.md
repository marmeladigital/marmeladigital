# Мармелад

Студия по разработке сайтов на Тильде.

Делаем лендинги, корпоративные сайты и digital-продукты для бизнеса. Прорабатываем структуру, UX, визуальную подачу и помогаем собирать сайты, которые выглядят сильно и решают коммерческие задачи.

## Что делаем

- разработка сайтов на Тильде
- лендинги и промостраницы
- корпоративные сайты
- проектирование структуры и прототипов
- дизайн интерфейсов
- кастомизация Zero Block
- базовая SEO-подготовка страниц

## Подход

Любим чистую структуру, понятную логику страницы и аккуратный дизайн без визуального шума. Для нас сайт — это рабочий инструмент бизнеса: он должен быть удобным, быстрым, убедительным и понятным для пользователя.
